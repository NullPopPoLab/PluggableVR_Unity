PluggableVR for EmotionCreators

EmotionCreators をVRで動作させるためのプラグインです。
HF Patch 3.12導入済の環境で動作確認しています。

使い方説明

対応VR環境

現状、OculusVR専用です。
(OpenVRでの対応に失敗した経緯ありまして)

導入手順

  1. 付属の BepInEx EmotionCreators_Data EmotionCreators_novr.bat EmotionCreators_vr.bat をゲームフォルダに上書き(というか追加だけのはず)します。
  2. EmotionCreators_Data/globalgamemanagers のVR版を作成します
    1. 起動時にも自動対応されますが、一応ここで globalgamemanagers のコピーを globalgamemanagers_novr にバックアップしておくと比較的安全です。
    2. ここ参考(手抜き)に、 globalgamemanagers を書き換えます。ただし、
      • globalgamemanagers 自体は書き換えないで、 globalgamemanagers_vr に生成します。
      • 修正後の方に違いあります
         0 vector enabledVRDevices
          0 Array Array (1 items)
           0 int size = 1
           [0]
            1 string data = "Oculus"

起動方法

  • EmotionCreators_vr.bat で起動するとVR onで動作
  • EmotionCreators_novr.bat で起動するとVR offで動作
  • 通常通り EmotionCreators.exe で起動すると前回の選択で

操作方法

全くの未実装で、カメラがルームスケールで動くだけです。
というか現状ではまともに動作しないです。
