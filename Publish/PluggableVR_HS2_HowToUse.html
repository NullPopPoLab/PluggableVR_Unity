PluggableVR for HoneySelect2

ハニーセレクト2 をVRで動作させるためのプラグインです。
HF Patch 3.12導入済の環境で動作確認しています。

使い方説明

対応VR環境

現状、OculusVR専用です。
(OpenVRでの対応に失敗した経緯ありまして)

導入手順

  1. 付属の BepInEx HoneySelect2_Data HoneySelect2_novr.bat HoneySelect2_vr.bat をゲームフォルダに上書き(というか追加だけのはず)します。
  2. HoneySelect2_Data/globalgamemanagers のVR版を作成します
    1. 起動時にも自動対応されますが、一応ここで globalgamemanagers のコピーを globalgamemanagers_novr にバックアップしておくと比較的安全です。
    2. ここ参考(手抜き)に、 globalgamemanagers を書き換えます。ただし、
      • globalgamemanagers 自体は書き換えないで、 globalgamemanagers_vr に生成します。
      • 修正後の方に違いあります
         0 vector enabledVRDevices
          0 Array Array (1 items)
           0 int size = 1
           [0]
            1 string data = "Oculus"

起動方法

  • HoneySelect2_vr.bat で起動するとVR onで動作
  • HoneySelect2_novr.bat で起動するとVR offで動作
  • 通常通り HoneySelect2.exe で起動すると前回の選択で

操作方法

  • VRヘッドセット
    • カメラをルームスケールで移動させます
  • 主スティック (右利き設定では左スティック)でカメラ位置マークを前後左右または上下左右に移動させます
    (主スティックを押し込むことで移動方法を切り替えます)
  • 副スティック (右利き設定では右スティック)でカメラ位置マークを左右に回転させます
  • 両方のスティックから指を離すことで、カメラを位置マーク地点に移します
  • あと、当面の調査用として、Yボタン押しながらBボタン押すと現在のHierarchy状態ごっそりファイルに書き出します
UI操作は通常通りPC画面のマウス操作で。

既知の問題点

  • UI未対応
  • 足元位置未対応のため、スティック回転の軸が怪しげ
  • なんか世界が巨大
  • えっち中は乗っ取ったVR用カメラとは別のカメラから映してるようでなんか遠い
